 MyPartyHostess - Discount Section

A dynamic and responsive Discount Management interface built using React, Vite, and Tailwind CSS. This component allows administrators to seamlessly manage, create, and edit promotional discounts for the MyPartyHostess platform.

 🚀 Features
* Discount Management: View, add, and organize ongoing platform discounts.
* Interactive UI: Smooth side drawers/modals for editing discount details.
* Modern Stack: Built on top of Vite for blazing-fast development and optimized production builds.
* Responsive Design: Fully optimized for desktop, tablet, and mobile layouts.

## 🛠️ Tech Stack
* Frontend: React.js (JSX)
* Build Tool: Vite
* Styling: Tailwind CSS (or standard CSS depending on configuration)
* Linting: ESLint
